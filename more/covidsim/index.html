Covid-19 Simulation
Eine Simulation der Ausbreitung einer Pandemie in C. Läuft aktuell nur auf einem Raspberry Pi 3B+ oder älter, der Pi 4 wird aufgrund des neuen Grafiktreibers noch nicht unterstützt. Aufgrund der tiefen Integration ins System durch die direkte Ansprechung des Framebuffers ist der Raspberry Pi der einzige Computer, der dieses Programm ausführen kann.

Es werden die täglichen Begenungen mit anderen Personen und die Beschränkungen durch verschiedene Lockdowns simuliert, um eine möglichst realitätsnahe Kurve zu generieren. Der Code für die Generierung von Textzeichen im Framebuffer und das Zeichnen von Formen wurde passend zur Anwendung geschrieben und muss bei der Kompilierung mit eingebunden werden.
